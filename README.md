### 介绍：

MVVM轻量快速开发框架：使用JetPack中的AndroidX + ViewModel +
LiveData + ViewBinding，同时使用ViewPager2 + RxJava3 + Retrofit2 +
Glide等主流框架进行搭建

### 使用步骤

- 下载本项目，在你的项目中点击`File —> New -> Import Module...`然后选择`basicLib`；然后在你的项目中依赖`basicLib`
